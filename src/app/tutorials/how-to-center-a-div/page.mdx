# How to Center a Div in CSS

Centering a div in CSS is a common task for web developers. There are multiple methods to achieve this, depending on whether you want to center the div horizontally, vertically, or both. This guide will walk you through various techniques to center a div using CSS.

<Custom />

## Centering Horizontally

### Using `margin: auto`

The `margin: auto` method is a straightforward way to center a block-level element horizontally.

```css
.container {
  width: 50%;
  margin: 0 auto;
  background-color: lightblue;
}
```
