# Color Formats in CSS

CSS gives us many options when it comes to expressing color—we can use hex codes, rgb, hsl, and more. Which option should we choose? This turns out to be a surprisingly important decision! In this article, we'll take a tour of color formats in CSS, and see which option will serve us best.

## Introduction

Color is a fundamental part of web design, and CSS provides several ways to specify color values. Understanding the differences and use cases for each format can help you make better decisions in your projects.

## Hex Codes

Hex codes are one of the most common ways to specify colors in CSS. They are compact and widely recognized.
